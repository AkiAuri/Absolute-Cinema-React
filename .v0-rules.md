# Project Rules and Preferences

## Language
- **NO TypeScript** - Use JavaScript (.js, .jsx) only
- All new files should be .js or .jsx, not .ts or .tsx

## Currency
- Use Philippine Peso sign (₱) for all currency displays
- Do not use dollar sign ($)

## Design
- Follow mobile-first design approach
- Use Tailwind CSS with semantic classes
- Maximize use of flexbox for layouts
- Keep color palette to 3-5 colors max
- Use design tokens via globals.css for consistency

## Code Organization
- Split into multiple components (no large monolithic files)
- Use existing patterns and utilities
- Remove imports when code is no longer used
- Use console.log("[v0] ...") for debugging only, remove after complete
